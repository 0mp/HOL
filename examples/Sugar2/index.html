Deep embedding of Sugar 2.0 in HOL

This directory contains a deep embedding of the Accellera standard property language Sugar 2.0 in HOL.

  • Category B submission to TPHOLs2002 entitled Using HOL to study Sugar 2.0 semantics
    [postscript | pdf ]

  • Script for theory of finite and infinite paths: PathScript.sml

  • Script for the semantics of Sugar 2.0: Sugar2SemanticsScript.sml

  • Script for results about Sugar 2.0 proved so far: Sugar2Script.sml
